# Endless-Runner-Game
Endless Runner Game is a 2D arcade-style game where the player continuously runs forward and avoids obstacles to achieve the highest possible score. The game ends when the player collides with an obstacle.

Endless Runner Game

Overview

Endless Runner Game is a 2D arcade game developed using Python and Pygame. The player controls a character that must jump over incoming obstacles while trying to survive as long as possible.

Features

* Endless gameplay
* Dynamic obstacle generation
* Jump controls
* Collision detection
* Score tracking

Technologies Used

* Python
* Pygame

Installation

pip install pygame

Run Project

python endless_runner.py

Learning Outcomes

* Game loops
* Event handling
* Collision detection
* Object movement
* Score systems
